# BrilhoFlyout — Controle de brilho via DDC/CI

Repositório oficial:  
https://github.com/MarcosVN007/BrilhoFlyout

Este app permite controlar o brilho de monitores externos pela bandeja do Windows.

## Como baixar o EXE
1. Vá na aba **Actions**
2. Clique no workflow mais recente
3. Desça até **Artifacts**
4. Baixe o arquivo **BrilhoFlyout.zip**
5. Dentro dele está o executável pronto

Ou acesse Releases quando eu criar automaticamente.

## Tecnologias
- WinUI 3
- .NET 8
- H.NotifyIcon
- MonitorControl (DDC/CI)
